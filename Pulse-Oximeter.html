Pulse Oximeter

For this project, I designed the software and hardware of a pulse oximeter.
I first created a program on LabVIEW that would alternatively switch the LEDs on and off,
and I then helped create a SOLIDWORKS model to create a fingerclip, cord, and case,
which I 3D-printed using an .STL model.

This is the presentation that contains the early SOLIDWORKS models of the fingerclip and box covering.
